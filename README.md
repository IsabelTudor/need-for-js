# Need For JS
Car Racing Game written in HTML, CCS and JavaScript.

## Gameplay
### Move using:
- `Left arrow ←`
- `Up arrow ↑`
- `Down arrow ↓`
- `Right arrow →`

## Demo
You can view a live version [here](https://tinawebdev.github.io/need-for-js/).
