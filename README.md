# Need For JS
Car Racing Game written in HTML, CCS and JavaScript.

## Gameplay
### Move using:
- `Up arrow ↑` or `W`
- `Left arrow ←` or `A`
- `Down arrow ↓` or `S`
- `Right arrow →` or `D`
